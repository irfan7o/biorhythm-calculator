# Biorhythm Calculator

This is the result of my [Ionic](https://ionicframework.com) course in Udemy by [Mirko Nasato](https://www.udemy.com/user/mirkonasato/)

How to run?
 * Open the project
 * Open the terminal in the menu bar Visual Studio Code
 * Type "npm start"
